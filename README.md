# AI Code Reviewer

A command-line tool that uses AI to review your code and provide detailed feedback.

## Features
- Paste code manually or load from a file
- AI-powered feedback on code quality, best practices, and improvements
- Formatted terminal output with syntax highlighting

## Tech Stack
- Python
- OpenRouter API (LLM integration)
- `rich` library for terminal UI
- `python-dotenv` for secure API key management
